git checkout <хеш коммита>

Позволяет нам перейти и посмотреть любой коммит

но так-же мы можем перейти и на любую существующую ветку

git checkout main

Не редко бывают случаи когда ты работаешь и хочешь закоммитить что-то, но тут
оказывается что ты забыл переключится на последний коммит от которого можно
создавать новые коммиты в текущей ветке.

Для того чтобы глянуть где именно мы находимся, можно использовать комманду
В нормальной ситуации мы будем на последнем коммите ветки main или другой своей ветки

git branch

> git branch

# О том, что такое main, мы поговорим позже
* main

Но если перед этим мы сделали git checkout на ранний коммит, то получим похожий вывод

> git branch

* (HEAD detached at <Хеш коммита>)
  main
